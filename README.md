# Cooking Recipe Chatbot with Voice Assistant

## Technologies Used:
- Flask + MongoDB Atlas
- React.js
- Google Speech-to-Text + Text-to-Speech


## Quick Start:



### 1. Run Backend:
```bash
cd backend
pip install -r requirements.txt
python app.py
```

### 3. Run Frontend:
```bash
cd frontend
npm install
npm start
```
